## How to Run

---

- <h3>Using yarn</h3>

  ```
  $ git clone https://github.com/worjs320/animation-website.git
  $ cd animation-website
  $ yarn install
  $ yarn start
  ```

- <h3>Using npm</h3>

  ```
  $ git clone https://github.com/worjs320/animation-website.git
  $ cd animation-website
  $ npm install
  $ npm start
  ```
